# Gamified Savings Vault - Savings System

This document describes the savings mechanism of the Algohack Vault.

## Core Idea
The gamified savings vault turns saving into an engaging experience:
- Users deposit ALGO into vault contracts.
- Deposits are associated with personalized savings goals.
- Users earn progress milestones or reward tiers.
- Interaction happens via a wallet-enabled React frontend.

## Key Features
1. **Deposit Funds**: Users can deposit ALGO into their vault.
2. **Track Goals**: Each deposit can contribute toward a specific savings goal.
3. **Earn Milestones**: Progress triggers visual milestones and reward notifications.
4. **Wallet Integration**: Users can connect Algorand wallets for deposits, withdrawals, and tracking.

## Notes
- The system is currently running on LocalNet by default.
- Can be extended to TestNet/MainNet with proper environment configuration.
- Future enhancements may include multiple vaults, automated reward tiers, and enhanced gamification features.
